# AI Code Understanding Agent

> **An AI-powered tool that reads any code file and explains it using diagrams, real-world analogies, and step-by-step walkthroughs — all generated by Google Gemini.**

---

## ✨ Features

| Feature | Description |
|---|---|
| 🤖 **AI Analysis** | Powered by Google Gemini 1.5 Flash |
| 🗺️ **Mermaid Diagrams** | Auto-generated flowcharts, class & sequence diagrams |
| 🌍 **Real-World Analogies** | Every concept explained with a relatable everyday example |
| 🚶 **Step-by-Step Walkthrough** | Traces the actual code execution flow |
| 💡 **Key Concepts** | Highlights important patterns and structures |
| 🚀 **Improvement Suggestions** | AI-generated tips to improve the code |
| 💻 **Rich Terminal UI** | Beautiful colored output in your terminal |
| 📄 **HTML Report** | Interactive report with syntax highlighting |

## 🚀 Quick Start

### 1. Clone / Open the project

```bash
cd "AI AGENT"
```

### 2. Set up environment

```bash
python -m venv .venv
source .venv/bin/activate    # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Add your Gemini API key

```bash
cp .env.example .env
```

Edit `.env` and paste your key:
```
GOOGLE_API_KEY=your_actual_key_here
```

> Get a free key at: https://aistudio.google.com/app/apikey

### 4. Run the agent

```bash
# Analyze any file interactively
python main.py

# Or pass a file directly
python main.py example_code.py

# Skip HTML report
python main.py myfile.py --no-html
```
